# Java 前置学习

## 为什么学习Java？
1. Java 具有丰富的社区 
2. Java 体系强大
3. 涉足广泛

## Java的特点
1. `面向对象`(对对象中的类、对象、继承、封装、多态、接口、包等均有很好的支持。)
2. 平台无关 (WriteOnce，RunanyWhere)
3. 简单性 (舍弃了很多C++中难以理解的特性，如操作符的重载和多继承等，而且Java语言不使用指针，加入了垃圾回收机制)
4. 解释执行
5. 多线程
6. 健壮性 (强类型机制、异常处理、垃圾回收机制等)
7. 安全性
8. 支持网络编程
...
